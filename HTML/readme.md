SOLUCION 


 1. Conceptos de Internet

a. Servidor DNS
Es un sistema que traduce nombres de dominio (como google.com) en direcciones IP para que el navegador pueda encontrar el servidor correcto.

b. IP
Es un número único que identifica un dispositivo en una red (como una dirección de casa en Internet).

c. Protocolo
Son reglas que permiten la comunicación entre dispositivos (por ejemplo, HTTP para páginas web).

d. Nombre de dominio
Es el nombre fácil de recordar de un sitio web (como facebook.com) que reemplaza la IP.

e. Directorio y nombre de archivo
Es la ruta dentro de un sitio web que indica dónde está un recurso específico (ej: /imagenes/foto.jpg).



2. Roles en desarrollo web

a. Diseñador UI/UX
Diseña cómo se ve (UI) y cómo se usa (UX) una aplicación para que sea atractiva y fácil de usar.

b. Cloud Manager / Ingeniero de infraestructura en la nube
Administra servidores, redes y servicios en la nube para que la app funcione de forma segura y eficiente.

c. Administrador de base de datos
Gestiona, organiza y protege los datos de la aplicación.

d. Project Manager / Director del Proyecto
Coordina el equipo, tiempos y recursos para que el proyecto se complete correctamente.



3. Lenguaje Turing completo

Significa que un lenguaje puede resolver cualquier problema computacional si tiene suficiente tiempo y memoria.



4. Por qué CSS y HTML no son lenguajes de programación

Porque no tienen lógica (como condiciones o ciclos complejos), solo sirven para estructura (HTML) y diseño (CSS).



5. Etiqueta en HTML

Es un elemento que define partes de una página web, usando palabras entre < > (ej: `<p>` para párrafos).
